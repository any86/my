# @any86/sleep
[![NPM Version][npm-image]][npm-url]

[npm-image]: https://badgen.net/npm/v/@any86/sleep
[npm-url]: https://npmjs.org/package/@any86/sleep

延迟, 模拟PHP的sleep函数

## 安装
```shell
npm i -S @any86/sleep
```

## 快速开始
```javascript
import sleep from '@any86/sleep';
// 延迟2秒
await sleep(2000);
```
